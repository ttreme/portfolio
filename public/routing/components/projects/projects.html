Troll-Patrol

Technologies Used: HTML, CSS, AngularJs, Node, Mongoose, MongoDb, Sockets.io, BootStrap
Troll-Patrol backstory: The idea from the project came from a conversation with friends that are always arguing. I figured I could make them a chat channel, and provide a "topic of the day" to argue over. Since there are so many trolls on social media and videogames, I thought, "Why not make a chat channel that is specifically for trolling?"
  • I used JWT for User Authentication in this project. This allows for users to receive a web token to verify they are logged in, and it helps protect the password!
  • For the chat I used Socket.io which allows for instant chats. It essentially listens for a message to be emitted , and then allows you to use the message instantly.
  • The rest is pretty standard!

Omelet-Haven

Technologies Used: HTML, CSS, AngularJs, Node, Mongoose, MongoDb, BootStrap
Omelet-Haven backstory: I love Omelets, so I figured I would make a goofy web application that could store omelets created by anybody! The website is very simple, but the idea is fun!

Last FM

Technologies Used: HTML, CSS, AngularJs, BootStrap
Last FM backstory: I simply wanted to be able to post music that I was listening to on Spotify to a website. The LastFm API made this easy to do, because it had straightforward documention to follow.
  • One page allows a user to enter their LastFm username, and will post the 10 most recent songs they have listened to.
  • The next page allows a user to pull the Top Ten Artists, and show how many listeners they have!
  • The last page is a very simple contact page with information to reach me!
